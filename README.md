# 🚆 TrainCloud.Microservices.Core

Shared features for all `TrainCloud.Microservices.*`.

* Controller
* Cache
* Linq (Sort/Paging)
* QuestPdf
* Swagger
* Global Api Filters
* Middleware
* MessageBus (Google Cloud Pub/Sub)
