# 🚆 TrainCloud.Microservices.Core

Shared features for all `TrainCloud.Microservices.*`.

* Kubernetes features
* Controller
* Cache
* Linq (Sort/Paging)
* QuestPdf
* Swagger
* Global Api Filters
* Middleware
* MessageBus (RabbitMq)
